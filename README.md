คู่มือการใช้งานบอท - Kai Hub PB
==================================================

ยินดีต้อนรับสู่ Kai Hub PB ครับ
นี่คือขั้นตอนการติดตั้งและใช้งานบอทแบบง่ายๆ สำหรับลูกค้าทุกท่าน

1. การเตรียมพร้อมก่อนใช้งาน (Installation)
   - ให้ทำการปิด Antivirus หรือ Windows Defender ชั่วคราว เพื่อป้องกันไม่ให้ไฟล์บอทถูกลบ
   - ดาวน์โหลดไฟล์บอทและแตกไฟล์ (Extract File) ลงในโฟลเดอร์ที่ต้องการ
   - *สำหรับผู้ใช้ Windows 7:* หากเข้าไม่ได้ ให้ดับเบิ้ลคลิกไฟล์ "Fix_Win7_TLS.reg" แล้วกด Yes เพื่อแก้ปัญหาการเชื่อมต่อ

2. การตั้งค่าเกมและตัวละคร (Game & Character Settings) **สำคัญมาก**
   เพื่อให้บอททำงานได้อย่างถูกต้อง กรุณาตั้งค่าตามนี้:
   - ความละเอียดหน้าจอ (Resolution): 1024 x 768
   - โหมดหน้าจอ (Window Mode): โหมดหน้าต่าง (Window Mode)
   - กราฟิก (Graphics): ปรับต่ำสุดทุกอย่าง (Low)
   - ความเร็วเมาส์ (Mouse Sensitivity): 0.100 (เฉพาะตัวหลัก/ตัวยิง)
   - ฉายา (Titles): ถอดออกให้หมด ห้ามใส่ฉายา

3. รูปแบบการใช้งาน (Bot Setup Strategy)
   - เครื่องหลัก (ตัวยิง): ให้ใช้บอทตัว "160 Kill"
   - ตัวไก่ (8 จอ): ให้ใช้บอทตัว "Corner" (บอทชิดมุม/ยืนนิ่ง)

4. การตั้งค่ารหัสผ่านและระบบออโต้ (Auto Login & Room Setup)
   - **ID/Pass:** กรอกไอดีและรหัสผ่านของตัวละครในช่องที่กำหนด เพื่อให้บอทล็อกอินเข้าเกมอัตโนมัติ
   - **Auto Login:** ติ๊กถูกที่ช่อง "Auto Login" หรือ "เข้าเกมอัตโนมัติ"
   - **Server/Channel:** เลือกเซิร์ฟเวอร์และห้องที่ต้องการให้บอทเข้า
   - **Room Password:** หากต้องการล็อกห้องส่วนตัว ให้ใส่รหัสผ่านห้องในช่อง "Room Pass" หรือ "รหัสห้อง" บอทจะสร้าง/เข้าห้องด้วยรหัสนี้

5. วิธีการใช้งาน (How to use)
   - คลิกขวาที่ตัวโปรแกรมบอท (.exe) แล้วเลือก "Run as administrator"
   - นำ Key ที่ได้รับ กรอกลงในช่อง License หรือ Login
   - ตั้งค่าต่างๆ (ตามข้อ 2 และ 4) ให้เรียบร้อย
   - กดปุ่ม "เริ่มทำงาน" (Start) แล้วเปิดหน้าเกมขึ้นมา (หรือให้บอทเปิดเองถ้าตั้ง Auto Login)
   - ปล่อยให้บอททำงานได้เลย!

6. ข้อมูลติดต่อและช่วยเหลือ (Contact Us)

   - หรือติดตามข่าวสารอัปเดตได้ที่หน้าเว็บ
